# 🌍 Earth-01: Humanity Simulator
> **"Can Humanity Survive Until 2100?"**

**Earth-01** is an AI-powered civilization simulation platform where you step into the role of a global leader. Balance economic growth, public health, environmental protection, food security, and social stability to navigate humanity safely from **2026 to 2100**.

---

## 🚀 Key Features

* 🤖 **Dynamic AI Scenarios:** AI-generated crises (pandemics, water shortages, recessions) that respond directly to your past decisions.
* ⚖️ **Realistic Trade-off Engine:** No simple "good" or "bad" answers. Every policy decision explains its complex socio-economic and ecological impacts.
* 🌱 **SDG Integration:** Built-in tracking of UN Sustainable Development Goals, focusing heavily on **SDG 2 (Zero Hunger)**, **SDG 3 (Good Health)**, and **SDG 6 (Clean Water)**.
* 📈 **Interactive Scoring:** Dynamic sustainability, economic, and stability meters that update with every simulated year.

---

## 🛠️ Tech Stack

* **Frontend:** React.js / Vite / Tailwind CSS
* **AI & Logic:** OpenAI API / Gemini API (for dynamic narrative and impact generation)
* **Deployment:** Vercel or Netlify

---

## 🎮 How to Play

1. **Launch the platform:** Start the simulation in the year 2026.
2. **Review Global Events:** Read the dynamic global event generated by the AI for that year.
3. **Draft Policies:** Choose from 4 strategic interventions.
4. **Analyze Impacts:** Review the AI-generated consequence projection detailing how your decision alters the path forward.
5. **Survive:** Reach the year 2100 with stable metrics to achieve the **✅ Humanity Survived** outcome.

---

## 🌍 UN Sustainable Development Goals (SDGs)
Earth-01 serves as an educational tool promoting **systems thinking** in sustainable development:
* **SDG 2 (Zero Hunger):** Simulating agricultural policy, food logistics, and crop failure.
* **SDG 3 (Good Health and Well-being):** Managing pandemic preparedness and healthcare infrastructure.
* **SDG 6 (Clean Water and Sanitation):** Decisions surrounding desalination, recycling, and water conservation.

## 📦 Installation & Setup

```bash
# Clone the repository
git clone [https://github.com/YOUR_USERNAME/earth-01.git](https://github.com/YOUR_USERNAME/earth-01.git)

# Navigate to project directory
cd earth-01

# Install dependencies
npm install

# Create a .env file and add your AI API keys
echo "VITE_AI_API_KEY=your_api_key_here" > .env

# Run locally
npm run dev
